1. Navigate to ..\ArtGalleryInventorySystem\

2. Compile: javac -d ../ArtGalleryInventorySystem AGIS_UI.java Artist.java Artwork.java Collage.java Drawing.java PerformanceArt.java PhysicalArt.java Print.java Painting.java Sculpture.java TwoDimensionalRendering.java

3. Run: java ArtGalleryInventorySystem.AGIS_UI

Tests are built in to the program. Input T at the main menu to access the testing suite.

AGIS_UI is the file that contains the main class and must be run in order to see the program in action
